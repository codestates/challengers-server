# challengers-server

## 프로젝트명 : challengers
## 팀 명 : CoDeMon

#### 프로젝트 내용
- 사용자가 자신의 도전(challenge)을 사람들과 공유하고 함께하는 서비스
- 개인이 도전을 생성하여 온라인에 업로드
- 업로드된 다양한 도전들을 유저들이 선택하여 도전해 볼 수 있다
- 많은 도전자들 중에 좋아요를 많이 받은 사람을 선택하여 명예의 전당에 표시
