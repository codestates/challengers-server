# challengers
사용자가 자신의 도전을 사람들과 공유하고 함께하는 서비스

## Team CoDeMon
## Project
21.04.26 ~ 21.05.07(2주)

## 기능구현 내용
- 회원가입 후 로그인 및 정보변경을 할 수 있다.
- 개인이 도전을 생성하여 온라인에 업로드를 할 수 있다.
- 업로드된 다양한 도전들을 유저들이 선택하여 도전해 볼 수 있다.
- 많은 도전자들 중에 좋아요를 많이 받은 사람을 선택하면 명예의 전당에 표시된다.
